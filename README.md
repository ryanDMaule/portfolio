Overview

This project is a personal portfolio website designed to showcase my work as both a software engineer and product-focused designer.

Rather than following a traditional portfolio template, the goal was to create an experience that felt personal, memorable, and representative of my own design preferences. The website combines large-scale typography, motion, visual experimentation, project case studies, and responsive layouts to create something that feels more like an interactive poster than a conventional portfolio.

The project was designed in Figma before being developed using HTML, CSS, and JavaScript.

⸻

Project Goals

The project aimed to:

* Create a memorable personal brand.
* Showcase software engineering projects.
* Demonstrate UI/UX and visual design skills.
* Explore animation and motion design.
* Build a fully responsive website.
* Create a portfolio that felt personal rather than template-driven.

Many portfolio websites communicate technical skills effectively but lack personality. This project was an attempt to present both technical ability and creative thinking within a single experience.

⸻

Features

Personal Branding

The website was designed around a strong visual identity centred on bold typography, geometric patterns, layered compositions, and large-scale visual elements.

The goal was to create something immediately recognisable rather than relying on a generic portfolio template.

Project Showcase

The portfolio highlights a number of personal projects including:

* Corner Pocket
* Escape the Dark Castle
* Portfolio Design Concepts

Each project includes descriptions, screenshots, supporting visuals, and technology summaries.

Design Exploration

A dedicated section showcases alternative concepts and iterations created during the design process.

This allows visitors to see how ideas evolved and demonstrates the importance of iteration within the design workflow.

Responsive Design

The website supports three primary screen sizes, each handled with its own layout considerations rather than relying solely on automatic scaling.

This ensured that content remained visually engaging and accessible across desktop, tablet, and mobile devices.

Motion & Interaction

The website includes a variety of interactive elements including:

* Animated transitions
* Scrolling effects
* Image galleries
* Carousels
* Hover interactions
* Dynamic content presentation

These elements were used to create a more engaging browsing experience while supporting the overall visual identity.

Contact System

The site includes a custom contact section allowing visitors to get in touch directly through the website.

Small pieces of humour were intentionally incorporated throughout the site to make the experience feel more human and approachable.

⸻

Technical Challenges

Balancing Creativity and Usability

One of the biggest challenges was creating a design that felt visually unique without becoming difficult to navigate.

The website intentionally embraces a more experimental layout while still maintaining a clear information hierarchy.

Responsive Layout Design

Many sections were designed as bespoke compositions rather than standard content blocks.

Ensuring these layouts adapted correctly across multiple screen sizes required significant testing and refinement.

Scalability

The project was originally built around a relatively small number of portfolio pieces.

As additional projects were completed, it became increasingly important to consider how future work could be integrated without compromising the original design language.

Performance

The website contains a large amount of visual content, typography, and animated elements.

Careful consideration was required to maintain smooth performance while preserving the intended experience.

Design Philosophy

The portfolio was heavily influenced by brutalist and editorial design principles.

Rather than prioritising minimalism, the goal was to create a visually rich experience that felt energetic, personal, and distinctive.

Large typography, layered layouts, motion, and strong contrast were used throughout the project to create a sense of identity and personality.

Many sections were intentionally designed to resemble poster layouts or magazine spreads rather than conventional web pages.

The project reflects a belief that a portfolio should communicate not only what someone has built, but also how they think.
